# <a name="create-a-new-environment"></a>Δημιουργία νέου περιβάλλοντος

Αφού [αγοράσει μια άδεια χρήσης συνδρομής για το Dynamics 365 Customer Insights](paid-license.md), ο καθολικός διαχειριστής του μισθωτή Microsoft 365 λαμβάνει ένα μήνυμα ηλεκτρονικού ταχυδρομείου που τον προσκαλεί να δημιουργήσει το περιβάλλον. Μεταβείτε στη σελίδα [https://home.ci.ai.dynamics.com/start](https://home.ci.ai.dynamics.com/start) για να ξεκινήσετε. Σε αυτό το σενάριο, ξεκινήστε με το [Βήμα 1: Παροχή βασικών πληροφοριών](#step-1-provide-basic-information).

Μετά τη δημιουργία του πρώτου περιβάλλοντος, ο καθολικός διαχειριστής του μισθωτή του Microsoft 365 μπορεί [να προσθέσει χρήστες από τον οργανισμό του ως διαχειριστές](permissions.md). Αυτοί οι διαχειριστές μπορούν να διαχειριστούν τους χρήστες και τα περιβάλλοντα. Εάν ο οργανισμός σας αγοράσει περισσότερες από μία άδειες χρήσης, για το Customer Insights, [επικοινωνήστε με την ομάδα υποστήριξής μας](https://go.microsoft.com/fwlink/?linkid=2079641) για να αυξήσετε τον αριθμό των διαθέσιμων περιβαλλόντων. Για περισσότερες πληροφορίες σχετικά με την χωρητικότητα και την ικανότητα διάθεσης πρόσθετων, ανατρέξτε στον [οδηγό άδειας χρήσης του Dynamics 365](https://go.microsoft.com/fwlink/?LinkId=866544). Αφού έχετε τη δυνατότητα να δημιουργήσετε πρόσθετα περιβάλλοντα, μεταβείτε στην επιλογή [Έναρξη της διεργασίας δημιουργίας περιβάλλοντος](#start-the-environment-creation-process).

> [!TIP]
> Αν θέλετε να δοκιμάσετε την υπηρεσία, ανατρέξτε στο θέμα [Ρύθμιση δοκιμαστικού περιβάλλοντος](trial-signup.md).

## <a name="prerequisites"></a>Προϋποθέσεις

[Δικαιώματα Διαχειριστή](permissions.md) στο Customer Insights

## <a name="start-the-environment-creation-process"></a>Έναρξη της διεργασίας δημιουργίας περιβάλλοντος

1. Ανοίξτε τον επιλογέα περιβάλλοντος και επιλέξτε **+ Δημιουργία**.
  
   :::image type="content" source="media/environment-picker.png" alt-text="Επιλέξτε τον επιλογέα περιβάλλοντος.":::

1. Ακολουθήστε την καθοδηγούμενη εμπειρία που περιγράφεται στις ακόλουθες ενότητες για να παράσχετε όλες τις απαιτούμενες πληροφορίες για ένα νέο περιβάλλον.

## <a name="step-1-provide-basic-information"></a>Βήμα 1: Παροχή βασικών πληροφοριών

1. Επιλέξτε αν θέλετε να δημιουργήσετε ένα περιβάλλον από την αρχή ή αν θα αντιγράψετε δεδομένα από ένα άλλο περιβάλλον. [Η αντιγραφή δεδομένων από ένα άλλο περιβάλλον](#copy-the-environment-configuration) απαιτεί πρόσθετα βήματα.

   :::image type="content" source="media/environment-settings-dialog.png" alt-text="Πλαίσιο διαλόγου για τη δημιουργία νέου περιβάλλοντος Customer Insights.":::

1. Δώστε τις παρακάτω λεπτομέρειες:

   - **Όνομα**: δώστε όνομα σε αυτό το περιβάλλον. Αυτό το πεδίο είναι ήδη συμπληρωμένο εάν αντιγράψετε από ένα υπάρχον περιβάλλον, αλλά μπορείτε να το αλλάξετε.
   - **Επιλογή της επιχείρησής σας**: Κύριο κοινό για το νέο περιβάλλον: μεμονωμένοι καταναλωτές (B2C) ή [επιχειρηματικοί λογαριασμοί](work-with-business-accounts.md) (B2B). Εάν ο οργανισμός σας δραστηριοποιείται κυρίως με μεμονωμένα άτομα, όπως ένας μεταπωλητής ή μια καφετέρια, επιλέξτε μεμονωμένα προϊόντα. Αν το κύριο κοινό σας είναι άλλες εταιρείες, όπως ένας κατασκευαστής αυτοκινήτων ή μια εταιρεία χαρτιού, επιλέξτε επιχειρηματικούς λογαριασμούς.
   - **Τύπος**: Τύπος περιβάλλοντος: παραγωγή ή περιβάλλον προστατευμένης εκτέλεσης. Τα περιβάλλοντα προστατευμένης εκτέλεσης δεν επιτρέπουν προγραμματισμένη ανανέωση δεδομένων και προορίζονται για προκαταρκτική υλοποίηση και δοκιμή. Τα περιβάλλοντα προστατευμένης εκτέλεσης χρησιμοποιούν το ίδιο κύριο κοινό με το περιβάλλον παραγωγής που είναι επιλεγμένο τη δεδομένη στιγμή.
   - **Περιοχή**: η περιοχή στην οποία αναπτύσσεται και φιλοξενείται η υπηρεσία. Για [να χρησιμοποιήσετε το δικό σας λογαριασμό Azure Data Lake Storage](own-data-lake-storage.md) ή [να συνδεθείτε σε έναν υπάρχοντα οργανισμό Microsoft Dataverse](customer-insights-dataverse.md), το περιβάλλον Customer Insights πρέπει να βρίσκεται στην ίδια περιοχή.

1. Επιλέξτε **Επόμενο**.

## <a name="step-2-configure-data-storage"></a>Βήμα 2: Ρύθμιση παραμέτρων χώρου αποθήκευσης δεδομένων

1. Επιλέξτε τη θέση αποθήκευσης των δεδομένων του Customer Insights:

   - **Χώρος αποθήκευσης του Customer Insights**: Η διαχείριση του χώρου αποθήκευσης δεδομένων γίνεται αυτόματα. Είναι η προεπιλεγμένη επιλογή και συνιστάται η χρήση αυτής της επιλογής, εκτός εάν υπάρχουν συγκεκριμένες απαιτήσεις για την αποθήκευση δεδομένων στο δικό σας λογαριασμό αποθήκευσης.
   - **Azure Data Lake Storage**: Καθορίστε τον δικό σας λογαριασμό Azure Data Lake Storage για την αποθήκευση των δεδομένων ώστε να έχετε πλήρη έλεγχο στη θέση αποθήκευσης των δεδομένων. Ακολουθήστε τα βήματα στην ενότητα [Χρήση του δικο σας λογαριασμού Azure Data Lake Storage](own-data-lake-storage.md).

   :::image type="content" source="media/data-storage-environment.png" alt-text="Επιλέξτε την προτιμώμενη επιλογή για αποθήκευση των δεδομένων σας.":::

1. Επιλέξτε **Επόμενο**.

## <a name="step-3-connect-to-microsoft-dataverse"></a>Βήμα 3: Σύνδεση με Microsoft Dataverse

Αν έχετε ένα περιβάλλον Dataverse, συνδεθείτε στο Customer Insights. Μοιραστείτε δεδομένα με το Dataverse για να το χρησιμοποιήσετε με επιχειρηματικές εφαρμογές που βασίζονται στο Dataverse, όπως εφαρμογές Dynamics 365 Marketing ή εφαρμογές βάσει μοντέλου στο Power Apps.

1. Ακολουθήστε τα βήματα στην καρτέλα [Εργασία με δεδομένα του Customer Insights στο Microsoft Dataverse](customer-insights-dataverse.md).

   :::image type="content" source="media/dataverse-provisioning.png" alt-text="κοινή χρήση δεδομένων με Microsoft Dataverse που ενεργοποιείται αυτόματα για νέα περιβάλλοντα.":::

1. Επιλέξτε **Επόμενο**.

## <a name="step-4-finalize-the-settings"></a>Βήμα 4: Ολοκλήρωση των ρυθμίσεων

Εξετάστε τις καθορισμένες ρυθμίσεις. Όταν όλα φαίνονται ολοκληρωμένα, επιλέξτε **Δημιουργία** για να ρυθμίσετε το περιβάλλον.

Για να αλλάξετε ορισμένες από τις ρυθμίσεις αργότερα, ανατρέξτε στην ενότητα [Διαχείριση περιβαλλόντων](manage-environments.md).

## <a name="work-with-your-new-environment"></a>Εργασία με το νέο σας περιβάλλον

Εξετάστε τα παρακάτω άρθρα για να σας βοηθήσουν να έχετε γρήγορα αποτελέσματα με τη ρύθμιση παραμέτρων του Customer Insights:

- [Προσθήκη περισσότερων χρηστών και ανάθεση αδειών](permissions.md).
- Πραγματοποιήστε [Λήψη των προελεύσεων δεδομένων σας](data-sources.md) και εκτελέστε τες με τη [διαδικασία ενοποίησης δεδομένων](data-unification.md) για να λάβετε [ενοποιημένα προφίλ πελατών](customer-profiles.md).
- [Εμπλουτίστε τα ενοποιημένα προφίλ πελατών](enrichment-hub.md) ή [εκτελέστε μοντέλα πρόβλεψης](predictions-overview.md).
- [Δημιουργήστε τμήματα](segments.md) για την ομαδοποίηση πελατών και [μέτρα](measures.md) για κριτική KPI.
- [Ρυθμίστε τις συνδέσεις](connections.md) και τις [εξαγωγές](export-destinations.md) για να επεξεργαστείτε υποσύνολα των δεδομένων σας σε άλλες εφαρμογές.

## <a name="copy-the-environment-configuration"></a>Αντιγραφή της διαμόρφωσης περιβάλλοντος

Ως διαχειριστής, εάν επιλέξετε να αντιγράψετε τη ρύθμιση παραμέτρων από ένα υπάρχον περιβάλλον, επιλέξτε από τη λίστα όλων των διαθέσιμων περιβαλλόντων στον οργανισμό σας.

:::image type="content" source="media/environment-settings-dialog.png" alt-text="Στιγμιότυπο οθόνης των επιλογών ρυθμίσεων στις ρυθμίσεις περιβάλλοντος.":::

Αντιγράφονται οι παρακάτω διαμορφώσεις:

- Προελεύσεις δεδομένων που εισάγονται μέσω Power Query
- Ρύθμιση παραμέτρων ενοποίησης δεδομένων
- Τμήματα
- Μετρήσεις
- Σχέσεις
- Δραστηριότητες
- Αναζήτηση και φιλτράρισμα ευρετηρίου
- Εξαγωγές
- Ανανέωση χρονοδιαγράμματος
- Εμπλουτισμοί
- Μοντέλα πρόβλεψης
- Αναθέσεις ρόλων

### <a name="set-up-a-copied-environment"></a>Ρύθμιση περιβάλλοντος που έχει αντιγραφεί

Όταν αντιγράφετε τη ρύθμιση παραμέτρων περιβάλλοντος, ένα μήνυμα επιβεβαίωσης εμφανίζεται όταν το περιβάλλον που έχει αντιγραφεί έχει δημιουργηθεί. Εκτελέστε τα παρακάτω βήματα για να επιβεβαιώσετε τα διαπιστευτήρια.

1. Επιλέξτε **Μετάβαση σε προελεύσεις δεδομένων** για να δείτε τη λίστα των προελεύσεων δεδομένων. Σε όλες τις προελεύσεις δεδομένων εμφανίζεται μια κατάσταση **Απαιτούνται διαπιστευτήρια**.

   :::image type="content" source="media/data-sources-copied.png" alt-text="Λίστα των προελεύσεων δεδομένων που έχουν αντιγραφεί και χρειάζονται έλεγχο ταυτότητας.":::

1. Επεξεργαστείτε τις προελεύσεις δεδομένων και καταχωρίστε τα διαπιστευτήρια για την ανανέωσή τους. Οι προελεύσεις δεδομένων από το φάκελο Common Data Model και το Dataverse πρέπει να δημιουργηθούν με μη αυτόματο τρόπο με το ίδιο όνομα όπως στο περιβάλλον προέλευσης.

1. Αφού ανανεώσετε τις προελεύσεις δεδομένων, μεταβείτε στην ενότητα **Δεδομένα** > **Ενοποίηση**. Εδώ θα βρείτε τις ρυθμίσεις από το περιβάλλον προέλευσης. Επεξεργαστείτε όπως απαιτείται ή επιλέξτε **Ενοποίηση** > **Ενοποίηση προφίλ πελατών και εξαρτήσεων** για να ξεκινήσετε τη διαδικασία ενοποίησης δεδομένων και να δημιουργήσετε την ενοποιημένη οντότητα πελάτη.

   > [!TIP]
   > Για λογαριασμούς και επαφές, επιλέξτε **Ενοποίηση λογαριασμών** > **Ενοποίηση προφίλ και εξαρτήσεις**.

1. Όταν ολοκληρωθεί η ενοποίηση των δεδομένων, μεταβείτε στην επιλογή **Μέτρα** και **Τμήματα** για να τα ανανεώσετε.

1. Μεταβείτε στη **Διαχείριση** > **Συνδέσεις** για να επανενεργοποιήσετε τις συνδέσεις στο νέο σας περιβάλλον.

1. Μεταβείτε στα **Δεδομένα** > **Εμπλουτισμός** και **Δεδομένα** > **Εξαγωγές** για να τις ενεργοποιήσετε ξανά.
