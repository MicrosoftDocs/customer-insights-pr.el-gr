# <a name="whats-new-in-the-audience-insights-capability-of-dynamics-365-customer-insights"></a>Τι νέο υπάρχει στη δυνατότητα πληροφοριών κοινού του Dynamics 365 Customer Insights



Σας ανακοινώνουμε με χαρά τις πιο πρόσφατες ενημερωμένες εκδόσεις! Αυτό το άρθρο συνοψίζει τις δυνατότητες της δημόσιας έκδοσης προεπισκόπησης, τις γενικές βελτιώσεις διαθεσιμότητας και τις ενημερώσεις δυνατοτήτων. Για να δείτε τα μακροπρόθεσμα σχέδια δυνατοτήτων, ρίξτε μια ματιά στο θέμα [Σχέδια κυκλοφορίας του Dynamics 365 και του Power Platform](/dynamics365/release-plans/).

Αναλαμβάνουμε την ανάπτυξη ενημερώσεων σε βάση ανά περιοχή. Έτσι, ορισμένες περιοχές μπορεί να δουν τις δυνατότητες πριν από άλλες. Εάν δεν καθοριστεί διαφορετικά, δεν χρειάζεται να κάνετε καμία ενέργεια και θα ενημερώσουμε την εφαρμογή αυτόματα χωρίς διακοπές λειτουργίας.

> [!TIP]
> Για υποβολή και ψήφο σε αιτήματα δυνατοτήτων και προτάσεις προϊόντων, μεταβείτε στην [πύλη ιδεών εφαρμογής Dynamics 365](https://experience.dynamics.com/ideas/categories/?forum=79a8c474-4e35-e911-a971-000d3a4f3343&forumName=Dynamics%20365%20Customer%20Insights).


## <a name="december-2021-updates"></a>Ενημερώσεις Δεκεμβρίου 2021

Οι ενημερώσεις του Δεκεμβρίου 2021 περιλαμβάνουν νέες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="forward-customer-insights-logs-to-azure-monitor"></a>Προώθηση αρχείων καταγραφής του Customer Insights στην Παρακολούθηση Azure

Το Customer Insights παρέχει μια άμεση ενοποίηση με το την Παρακολούθηση Azure. Αυτή η δυνατότητα περιλαμβάνει συμβάντα ελέγχου και συμβάντα λειτουργίας. Τα αρχεία καταγραφής πόρων Παρακολούθησης Azure σάς επιτρέπουν να παρακολουθείτε και να στέλνετε αρχεία καταγραφής στον Χώρο αποθήκευσης Azure, στην Ανάλυση αρχείων καταγραφής Azure ή να τα διοχετεύετε στα Κέντρα συμβάντων Azure.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Προώθηση αρχείων καταγραφής στο Dynamics 365 Customer Insights με την Παρακολούθηση Azure (έκδοση προεπισκόπησης).](diagnostics.md).

### <a name="enrich-customer-profiles-with-engagement-data"></a>Εμπλουτισμός προφίλ πελατών με δεδομένα δέσμευσης

Χρησιμοποιήστε δεδομένα από το Microsoft Office 365 για να εμπλουτίσετε τα προφίλ λογαριασμών πελατών σας με πληροφορίες σχετικά με τις δεσμεύσεις μέσω εφαρμογών Office 365. Τα δεδομένα δέσμευσης αποτελούνται από δραστηριότητα ηλεκτρονικού ταχυδρομείου και σύσκεψης, η οποία συγκεντρώνεται σε επίπεδο λογαριασμού. Για παράδειγμα, ο αριθμός μηνυμάτων ηλεκτρονικού ταχυδρομείου από έναν επιχειρηματικό λογαριασμό ή ο αριθμός των συσκέψεων με το λογαριασμό. Δεν γίνεται κοινή χρήση δεδομένων σχετικά με μεμονωμένους χρήστες. Αυτός ο εμπλουτισμός διατίθεται στις εξής περιοχές: ΗΒ, Ευρώπη, Βόρεια Αμερική.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εμπλουτισμός προφίλ πελατών με δεδομένα δέσμευσης (έκδοση προεπισκόπησης)](enrichment-office.md)

### <a name="advanced-data-unification-features"></a>Δυνατότητες σύνθετης ενοποίησης δεδομένων

#### <a name="enable-conflict-resolution-policies-at-the-individual-attribute-level"></a>Ενεργοποίηση πολιτικών επίλυσης διενέξεων σε επίπεδο μεμονωμένου χαρακτηριστικού

Όταν καταργείτε διπλότυπες καρτέλες πελατών εντός μιας οντότητας, μπορεί να μην θέλετε να επιλέξετε μια πλήρη καρτέλα ως νικητή. Τώρα σας επιτρέπουμε να συγχωνεύετε τα καλύτερα πεδία από διάφορες καρτέλες βάσει κανόνων για κάθε χαρακτηριστικό. Για παράδειγμα, μπορείτε να επιλέξετε να διατηρήσετε την πιο πρόσφατη διεύθυνση ηλεκτρονικού ταχυδρομείου ΚΑΙ την πιο ολοκληρωμένη διεύθυνση από διαφορετικές καρτέλες. 

Τώρα μπορείτε να καθορίσετε ξεχωριστούς κανόνες συγχώνευσης για μεμονωμένα χαρακτηριστικά, ενώ καταργείτε διπλότυπες και συγχωνεύετε καρτέλες εντός μίας οντότητας. Προηγουμένως, σας είχαμε δώσει τη δυνατότητα να επιλέξετε μόνο έναν κανόνα συγχώνευσης (διατήρηση καρτελών με βάση την πληρότητα δεδομένων επάρκειας) και αυτός ο κανόνας εφαρμόστηκε στο επίπεδο καρτέλας σε όλα τα χαρακτηριστικά. Αυτό δεν είναι ιδανικό όταν ορισμένα από τα δεδομένα που θέλετε να διατηρήσετε βρίσκονται στην καρτέλα Α και άλλα καλά δεδομένα που βρίσκονται στην καρτέλα Β.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ορισμός κατάργησης διπλοτύπων σε μια οντότητα αντιστοίχισης](match-entities.md#define-deduplication-on-a-match-entity).

#### <a name="custom-rules-for-matching"></a>Προσαρμοσμένοι κανόνες για αντιστοίχιση

Υπάρχουν φορές που πρέπει να καθορίσετε μια εξαίρεση στους γενικούς κανόνες, προκειμένου να ΜΗΝ αντιστοιχιστούν καρτέλες. Αυτό μπορεί να συμβεί όταν πολλά άτομα μοιράζονται αρκετές πληροφορίες, έτσι ώστε το σύστημα να τα ταιριάζει με ένα άτομο. Για παράδειγμα, δίδυμα και το ίδιο επίθετο, που ζουν στην ίδια πόλη και μοιράζονται την ίδια ημερομηνία γέννησης.

Με τις εξαιρέσεις εξασφαλίζεται ότι είναι δυνατό να αντιμετωπιστεί εσφαλμένη ενοποίηση δεδομένων στους κανόνες ενοποίησης. Μπορείτε να προσθέσετε πολλές εξαιρέσεις σε έναν κανόνα.

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Προσθήκη εξαιρέσεων σε έναν κανόνα](match-entities.md#add-exceptions-to-a-rule).

#### <a name="provide-additional-conflict-resolution-policies-and-enable-grouping-of-attributes"></a>Παροχή πρόσθετων πολιτικών επίλυσης διενέξεων και ενεργοποίηση της ομαδοποίησης χαρακτηριστικών

Αυτή η δυνατότητα σάς επιτρέπει να αντιμετωπίζετε μια ομάδα πεδίων ως μεμονωμένη μονάδα. Για παράδειγμα, όταν οι καρτέλες μας περιέχουν τα πεδία Διεύθυνση1, Διεύθυνση2, Πόλη, Πολιτεία και Τ.Κ. Είναι πιθανό να μην θέλουμε να συγχωνευτούν στη Διεύθυνση2 διαφορετικής καρτέλας, νομίζοντας ότι θα είναι πιο ολοκληρωμένα τα δεδομένα μας.

Τώρα μπορείτε να συνδυάσετε μια ομάδα σχετιζόμενων πεδίων και να εφαρμόσετε μια μεμονωμένη πολιτική συγχώνευσης στην ομάδα. 

Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Συνδυασμός ομάδας πεδίων](merge-entities.md#combine-a-group-of-fields).

Οι εφαρμογές ενεργειών Dynamics 365 και οι εφαρμογές βάσει μοντέλου μπορούν να χρησιμοποιήσουν αυτά τα δεδομένα για να αναζητήσουν τις λεπτομέρειες της ιδιότητας μέλους τμήματος για ένα δεδομένο πελάτη.

## <a name="october-2021-updates"></a>Ενημερώσεις Οκτωβρίου 2021

Οι ενημερώσεις του Οκτωβρίου 2021 περιλαμβάνουν νέες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="b-to-b"></a>B2B

Από τον Οκτώβριο του 2021, μπορείτε να εργαστείτε με επιχειρηματικούς λογαριασμούς και τις σχετικές επαφές τους στο Customer Insights. Πριν, η εφαρμογή ήταν ως επί το πλείστον προσαρμοσμένη σε μεμονωμένες ρυθμίσεις. Ορισμένα πεδία δυνατοτήτων ενημερώθηκαν για την υποστήριξη σεναρίων B2B που περιλαμβάνει ο νέος τύπος περιβάλλοντος. Για μια επισκόπηση σχετικά με τις υποστηριζόμενες δυνατότητες B2B, ανατρέξτε στο θέμα [Εργασία με επιχειρηματικούς λογαριασμούς πληροφοριών κοινού](work-with-business-accounts.md).

Οι ακόλουθες ενότητες τονίζουν ορισμένους από τους βασικούς τομείς που προσαρμόστηκαν για την υποστήριξη επιχειρηματικών λογαριασμών και την προσαρμογή μεμονωμένων λογαριασμών.

#### <a name="export-segments-based-on-business-accounts"></a>Εξαγωγή τμημάτων της αγοράς με βάση επιχειρηματικούς λογαριασμούς

Όλες οι εξαγωγές τμήματος κοινό πληροφορίες είναι διαθέσιμες στο πλαίσιο των επιχειρηματικών λογαριασμών. Οι περισσότερες εξαγωγές τμήματος απαιτούν επιπλέον πληροφορίες ρύθμισης παραμέτρων και [επαφών που προβάλλονται](segment-builder.md#create-a-new-segment) στα υποκείμενα τμήματα ώστε να είναι έγκυρες για επιχειρηματικούς λογαριασμούς. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εξαγωγή τμημάτων](export-destinations.md#export-segments).

#### <a name="use-the-linkedin-ads-export-with-business-accounts"></a>Χρήση της εξαγωγής διαφημίσεων LinkedIn με επιχειρηματικούς λογαριασμούς

Η εξαγωγή διαφημίσεων LinkedIn είναι πλέον διαθέσιμη για στόχευση σε επαφές και εταιρείες στο πλαίσιο των επιχειρηματικών λογαριασμών. Όταν επιλέγετε τη στόχευση εταιρείας ως το κύριο σημείο εστίασης της εξαγωγής LinkedIn, μπορείτε να εξαγάγετε τμήματα της αγοράς που είναι ενσωματωμένα σε επιχειρηματικούς λογαριασμούς χωρίς να χρειάζεται να χρησιμοποιήσετε πληροφορίες επικοινωνίας έργου. Για περισσότερες πληροφορίες, μεταβείτε στα έγγραφα σχετικά με την [εξαγωγή διαφημίσεων LinkedIn](export-linkedin-ads.md) και τη διαφορά μεταξύ της [στόχευσης επαφών](https://business.linkedin.com/marketing-solutions/ad-targeting/contact-targeting) και της [στόχευσης εταιρείας](https://business.linkedin.com/marketing-solutions/ad-targeting/account-targeting). 

#### <a name="create-measures-based-on-business-accounts-and-their-hierarchy"></a>Δημιουργία ενεργειών βάσει επιχειρηματικών λογαριασμών και της ιεραρχίας τους

Το εργαλείο δόμησης μέτρησης σάς επιτρέπει να δημιουργείτε μέτρα σχετικά με τους επιχειρηματικούς λογαριασμούς και προαιρετικά να χρησιμοποιείτε τις πληροφορίες της ιεραρχίας. Οι πληροφορίες ιεραρχίας χρησιμοποιούνται για τη συνάθροιση ενός υπολογισμού μέτρησης σε ένα λογαριασμό και το μόνο που σχετίζονται με τους δευτερεύοντες λογαριασμούς. Για παράδειγμα, μπορείτε να δημιουργήσετε μέτρα, όπως τα συνολικά έσοδα, για κάθε ομάδα επιχειρηματικών λογαριασμών που έχει αναγνωριστεί από την ιεραρχία τους. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ορισμός και διαχείριση μέτρων](measures.md).

#### <a name="create-segments-based-on-business-accounts-and-their-hierarchy"></a>Δημιουργία τμημάτων βάσει επιχειρηματικών λογαριασμών και της ιεραρχίας τους

Το εργαλείο δόμησης τμημάτων σάς δίνει τη δυνατότητα να δημιουργήσετε τμήματα που προαιρετικά περιλαμβάνουν πληροφορίες επικοινωνίας για κάθε λογαριασμό σε ένα τμήμα. Εάν έχετε ορίσει ιεραρχία λογαριασμών, μπορείτε να χρησιμοποιήσετε πληροφορίες ιεραρχίας λογαριασμών στη δημιουργία τμήματος. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Δημιουργία νέου τμήματος](segment-builder.md#create-a-new-segment).

#### <a name="retain-your-business-accounts-with-deep-insights-to-their-churn-tendency"></a>Διατηρήστε τους επιχειρηματικούς λογαριασμούς σας με σημαντικές πληροφορίες σχετικά με την τάση απώλειάς τους

Το μοντέλο ανάπτυξης πρόβλεψη πελατών υποστηρίζει πλέον και τους επιχειρηματικούς λογαριασμούς. Μπορείτε να αξιολογήσετε τον κίνδυνο της εμφάνισης όχι μόνο για ένα λογαριασμό αλλά για ένα συνδυασμό λογαριασμού και μιας κατηγορίας προϊόντων ή υπηρεσιών που αγοράζουν από εσάς. Αυτή η προσθήκη σας βοηθά να κατανοήσετε εάν ένας λογαριασμός είναι πιο πιθανό να σταματήσει την αγορά από εσάς γενικά ή απλώς για μια συγκεκριμένη κατηγορία αγαθών ή υπηρεσιών. Για να σας βοηθήσει περαιτέρω να χρησιμοποιήσετε αυτό το μοντέλο AI, παραθέτει επίσης τους λόγους για τους οποίους είναι πιθανό να γίνει άτωτη ένας λογαριασμός. Για περισσότερες πληροφορίες, ανατρέξτε στην [πρόβλεψη συναλλαγών (προεπισκόπηση)](predict-transactional-churn.md).

#### <a name="see-contacts-of-a-business-account-in-customer-view"></a>Προβολή επαφών ενός επιχειρηματικού λογαριασμού στην προβολή "Πελάτης"

Εάν οι επιχειρηματικοί λογαριασμοί αντιστοιχίζονται σε σχετικούς λογαριασμούς, η εφαρμογή Customer Insights εμφανίζει αυτές τις σχετικές επαφές ως μέρος της προβολής λεπτομερειών πελάτη. Για περισσότερες πληροφορίες, συμβουλευτείτε τα [Προφίλ πελατών](customer-profiles.md).


## <a name="september-2021-updates"></a>Ενημερώσεις Σεπτεμβρίου 2021

Οι ενημερώσεις του Σεπτεμβρίου 2021 περιλαμβάνουν νέες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="activities"></a>Δραστηριότητες

- **Βελτιώσεις στο χρονοδιάγραμμα δραστηριοτήτων** Έχουμε επεκτείνει τα φίλτρα για το χρονοδιάγραμμα δραστηριοτήτων στα προφίλ πελατών. Επιπλέον, μπορείτε να χρησιμοποιήσετε το νέο παράθυρο φίλτρου για να φιλτράρετε κατά τύπο δραστηριότητας και κατά ημερομηνία. Μπορείτε να φιλτράρετε τις ημερομηνίες χρησιμοποιώντας διάφορες συνθήκες. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Προβολή χρονοδιαγραμμάτων δραστηριοτήτων σε προφίλ πελατών](activities.md#view-activity-timelines-on-customer-profiles).

### <a name="relationships"></a>Σχέσεις

- **Υποστήριξη σχέσεων πολλαπλής μεταπήδησης** Χρησιμοποιήστε σχέσεις πολλαπλής μεταπήδησης κατά τη ρύθμιση παραμέτρων δραστηριοτήτων και τον ορισμό σχέσεων μεταξύ οντοτήτων. Οι πολλαπλές σχέσεις χρησιμοποιούν μια ενδιάμεση οντότητα για τη σύνδεση δύο οντοτήτων. Όταν ρυθμίζετε τις παραμέτρους μιας δραστηριότητας, μπορείτε να χρησιμοποιήσετε μια σχέση πολλαπλής μεταπήδησης για να συνδέσετε την οντότητα της δραστηριότητας σε μια ενδιάμεση οντότητα και, στη συνέχεια, σε μια οντότητα πελάτη. Μπορείτε να συνδυάσετε σχέσεις πολλαπλής μεταπήδησης με σχέσεις πολλαπλών διαδρομών. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Σχέση πολλαπλής μεταπήδησης](relationships.md#multi-hop-relationship).

- **Υποστήριξη σχέσεων πολλαπλών διαδρομών** Χρησιμοποιήστε σχέσεις πολλαπλών διαδρομών κατά τη ρύθμιση παραμέτρων δραστηριοτήτων και τον ορισμό σχέσεων μεταξύ οντοτήτων. Οι οντότητες πολλών διαδρομών συσχετίζουν μια οντότητα προέλευσης με περισσότερες από μία οντότητες. Όταν ρυθμίζετε τις παραμέτρους μιας δραστηριότητας, μπορείτε να χρησιμοποιήσετε μια σχέση πολλαπλών διαδρομών για να συνδέσετε την οντότητα της δραστηριότητας σε περισσότερες από μια οντότητες πελάτη. Μπορείτε να συνδυάσετε σχέσεις πολλαπλών διαδρομών με σχέσεις πολλαπλής μεταπήδησης. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Σχέση πολλαπλών διαδρομών](relationships.md#multi-path-relationship).

   Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Ρύθμιση παραμέτρων φίλτρων προϊόντων](predict-product-recommendation.md#configure-product-filters).

- **Δημιουργία τμημάτων από προβλέψεις μοντέλων** Έχουμε προσθέσει έναν γρήγορο τρόπο για να δημιουργούμε τμήματα χρησιμοποιώντας τα αποτελέσματα ενός μοντέλου πρόβλεψης. Από τη σελίδα αποτελεσμάτων μοντέλου, μπορείτε εύκολα να δημιουργήσετε ένα νέο τμήμα επιλέγοντας **Δημιουργία τμήματος**.    
  Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Δημιουργία τμήματος με βάση ένα μοντέλο πρόβλεψης](prediction-based-segment.md).

- **Επεξηγήσεις για συστάσεις προϊόντων** Έχουμε προσθέσει πληροφορίες που εξηγούν τους βασικούς παράγοντες τους οποίους έχει μάθει το μοντέλο AI για τη δημιουργία προτάσεων προϊόντων και τον βαθμό στον οποίο οι παράγοντες αυτοί συνεισφέρουν στις προτάσεις προϊόντων. Αυτές οι πληροφορίες προστίθενται στην οθόνη αποτελεσμάτων μοντέλου.    
   Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Επανεξέταση μιας κατάστασης πρόβλεψης και αποτελέσματα](predict-product-recommendation.md#review-a-prediction-status-and-results).

## <a name="february-2021-updates"></a>Ενημερώσεις Φεβρουαρίου 2021

Οι ενημερώσεις του Φεβρουαρίου 2021 περιλαμβάνουν διάφορες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

#### <a name="extensibility"></a>Επεκτασιμότητα

- **Εξαγωγή τμημάτων στο AdRoll**

  Έχουμε επεκτείνει τους προορισμούς εξαγωγής για να συμπεριλάβουμε το AdRoll. Τώρα μπορείτε να εξαγάγετε τμήματα σε κοινά από το Customer Insights στο AdRoll και να τα χρησιμοποιήσετε ως γραμμή βάσης για τις διαφημίσεις σας. Για περισσότερες πληροφορίες, ανατρέξτε στη [Σύνδεση για AdRoll](export-adroll.md).

#### <a name="segments"></a>Τμήματα
 
- **Δημιουργία διπλοτύπου ενός τμήματος**
  
  Για να δημιουργήσετε ένα νέο τμήμα με βάση ένα υπάρχον, μπορείτε τώρα να δημιουργήσετε διπλότυπο τμήματος και να επεξεργαστείτε το διπλότυπο τμήμα για να το βελτιώσετε περαιτέρω. 

- **Προσθήκη επιπλέον χαρακτηριστικών σε τμήμα**

  Τώρα μπορείτε να συμπεριλάβετε χαρακτηριστικά στην έξοδο του τμήματός σας, ακόμα και αν αυτά τα χαρακτηριστικά δεν αποτελούν μέρος του προφίλ πελάτη. Για παράδειγμα, συμπεριλάβετε αναγνωριστικά συνδρομής σε ένα τμήμα, παρόλο που είναι μέρος της οντότητας συνδρομής που έχει σχέση M:1 με την οντότητα πελάτη. Εφόσον το χαρακτηριστικό ανήκει σε μια οντότητα που σχετίζεται με την οντότητα πελάτη, μπορείτε τώρα να συμπεριλάβετε αυτά τα χαρακτηριστικά.  

#### <a name="predictions"></a>Προβλέψεις

- **Δημιουργία προγνωστικών προτάσεων προϊόντων**

  Η κατανόηση του τι ενδιαφέρονται οι πελάτες να αγοράσουν είναι ένα από τα πρώτα βήματα που απαιτούνται για τη βελτίωση των επιχειρηματικών εσόδων και την ενίσχυση της αφοσίωσης των πελατών μέσω της εξατομίκευσης και της δέσμευσης. Η παροχή προτάσεων για προϊόντα που δεν είναι στοιχισμένα με τα ενδιαφέροντο του πελάτη σας μπορεί να δημιουργήσει μια αίσθηση αποσύνδεσης μεταξύ του πελάτη και της επιχείρησής σας και, εντέλει, να περιορίσει τα συνολικά πιθανά έσοδα και εμπειρία για έναν πελάτη. 

  Χρησιμοποιώντας τα δικά σας δεδομένα, μπορείτε πλέον να δημιουργήσετε προβλέψεις για τα προϊόντα που είναι πιθανό να αγοράσουν οι πελάτες σας στο μέλλον. Για περισσότερες πληροφορίες, ανατρέξτε στην [πρόβλεψη πρότασης προϊόντων](predict-product-recommendation.md).

#### <a name="system-administration"></a>Διαχείριση συστήματος

- **Το περιβάλλον αντιγραφής υποστηρίζει περισσότερους τύπους προέλευσης δεδομένων**

  Οι διαχειριστές μπορούν να αντιγράφουν ρυθμίσεις παραμέτρων περιβάλλοντος σε ένα νέο περιβάλλον στον ίδιο οργανισμό. Αυτή η δυνατότητα επεκτείνει τη λειτουργικότητα του περιβάλλοντος αντιγραφής για περιπτώσεις στις οποίες χρησιμοποιούνται οι προελεύσεις δεδομένων που βασίζονται σε διαχειριζόμενη λίμνη δεδομένων Microsoft Dataverse ή σε φάκελο Common Data Model.

Το σύστημα χρησιμοποιεί ένα μοντέλο AI για να προτείνει τμήματα που μπορούν να βοηθήσουν στη βελτίωση ενός KPI (δείκτη) που παρακολουθείτε ήδη. Δείχνουμε την έκταση της επιρροής των χαρακτηριστικών που επιλέγετε σε ένα μέγεθος ή σε άλλο πρωτεύον χαρακτηριστικό. - **Βελτιωμένη εμπειρία αντιστοίχισης** Στον τομέα ενοποίησης δεδομένων, η εμπειρία αντιστοίχισης ενημερώθηκε. Σας επιτρέπει να ρυθμίσετε τις παραμέτρους και να προβάλετε τους κανόνες αντιστοίχισης, συμπεριλαμβανομένων λεπτομερών στατιστικών στοιχείων για να εξηγήσετε περαιτέρω πώς λειτουργεί η αντιστοίχιση. Υπάρχουν επιλογές για την απενεργοποίηση ενός κανόνα αντιστοίχισης, ώστε να μην είναι πλέον ενεργός κατά τη διατήρηση των κανόνων ρύθμισης παραμέτρων, μεταφοράς και απόθεσης και πολλών άλλων.
