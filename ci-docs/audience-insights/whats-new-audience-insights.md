# <a name="whats-new-in-the-audience-insights-capability-of-dynamics-365-customer-insights"></a>Τι νέο υπάρχει στη δυνατότητα πληροφοριών κοινού του Dynamics 365 Customer Insights

[!INCLUDE [cc-data-platform-banner](../includes/cc-data-platform-banner.md)]

Σας ανακοινώνουμε με χαρά τις πιο πρόσφατες ενημερωμένες εκδόσεις! Αυτό το άρθρο συνοψίζει τις δυνατότητες της δημόσιας έκδοσης προεπισκόπησης, τις γενικές βελτιώσεις διαθεσιμότητας και τις ενημερώσεις δυνατοτήτων. Για να δείτε τα μακροπρόθεσμα σχέδια δυνατοτήτων, ρίξτε μια ματιά στο θέμα [Σχέδια κυκλοφορίας του Dynamics 365 και του Power Platform](/dynamics365/release-plans/).

Αναλαμβάνουμε την ανάπτυξη ενημερώσεων σε βάση ανά περιοχή. Έτσι, ορισμένες περιοχές μπορεί να δουν τις δυνατότητες πριν από άλλες. Εάν δεν καθοριστεί διαφορετικά, δεν χρειάζεται να κάνετε καμία ενέργεια και θα ενημερώσουμε την εφαρμογή αυτόματα χωρίς διακοπές λειτουργίας.

> [!TIP]
> ια υποβολή και ψήφο σε αιτήματα δυνατοτήτων και προτάσεις προϊόντων, μεταβείτε στην [πύλη ιδεών εφαρμογής Dynamics 365](https://experience.dynamics.com/ideas/categories/?forum=79a8c474-4e35-e911-a971-000d3a4f3343&forumName=Dynamics%20365%20Customer%20Insights).

## <a name="september-2021-updates"></a>Ενημερώσεις Σεπτεμβρίου 2021

Οι ενημερώσεις του Σεπτεμβρίου 2021 περιλαμβάνουν νέες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="activities"></a>Δραστηριότητες

- **Βελτιώσεις στο χρονοδιάγραμμα δραστηριοτήτων** Έχουμε επεκτείνει τα φίλτρα για το χρονοδιάγραμμα δραστηριοτήτων στα προφίλ πελατών. Επιπλέον, μπορείτε να χρησιμοποιήσετε το νέο παράθυρο φίλτρου για να φιλτράρετε κατά τύπο δραστηριότητας και κατά ημερομηνία. Μπορείτε να φιλτράρετε τις ημερομηνίες χρησιμοποιώντας διάφορες συνθήκες. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Προβολή χρονοδιαγραμμάτων δραστηριοτήτων σε προφίλ πελατών](activities.md#view-activity-timelines-on-customer-profiles).

### <a name="relationships"></a>Σχέσεις

- **Υποστήριξη σχέσεων πολλαπλής μεταπήδησης** Χρησιμοποιήστε σχέσεις πολλαπλής μεταπήδησης κατά τη ρύθμιση παραμέτρων δραστηριοτήτων και τον ορισμό σχέσεων μεταξύ οντοτήτων. Οι πολλαπλές σχέσεις χρησιμοποιούν μια ενδιάμεση οντότητα για τη σύνδεση δύο οντοτήτων. Όταν ρυθμίζετε τις παραμέτρους μιας δραστηριότητας, μπορείτε να χρησιμοποιήσετε μια σχέση πολλαπλής μεταπήδησης για να συνδέσετε την οντότητα της δραστηριότητας σε μια ενδιάμεση οντότητα και, στη συνέχεια, σε μια οντότητα πελάτη. Μπορείτε να συνδυάσετε σχέσεις πολλαπλής μεταπήδησης με σχέσεις πολλαπλών διαδρομών. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Σχέση πολλαπλής μεταπήδησης](relationships.md#multi-hop-relationship).

- **Υποστήριξη σχέσεων πολλαπλών διαδρομών** Χρησιμοποιήστε σχέσεις πολλαπλών διαδρομών κατά τη ρύθμιση παραμέτρων δραστηριοτήτων και τον ορισμό σχέσεων μεταξύ οντοτήτων. Οι οντότητες πολλών διαδρομών συσχετίζουν μια οντότητα προέλευσης με περισσότερες από μία οντότητες. Όταν ρυθμίζετε τις παραμέτρους μιας δραστηριότητας, μπορείτε να χρησιμοποιήσετε μια σχέση πολλαπλών διαδρομών για να συνδέσετε την οντότητα της δραστηριότητας σε περισσότερες από μια οντότητες πελάτη. Μπορείτε να συνδυάσετε σχέσεις πολλαπλών διαδρομών με σχέσεις πολλαπλής μεταπήδησης. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Σχέση πολλαπλών διαδρομών](relationships.md#multi-path-relationship).

## <a name="june-2021-updates"></a>Ενημερώσεις Ιουνίου 2021

Οι ενημερώσεις του Ιουνίου 2021 περιλαμβάνουν διάφορες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="data-ingestion"></a>Κατάποση δεδομένων

- **Βελτιωμένες ενημερώσεις της προόδου ενοποίησης δεδομένων** Μπορείτε τώρα να δείτε πιο λεπτομερείς και βελτιωμένες δυναμικές ενημερώσεις κατάστασης στα βήματα της [διαδικασίας ενοποίησης δεδομένων](data-unification.md). Αυτή η δυνατότητα σάς επιτρέπει να παρακολουθήσετε την αναλυτική πρόοδο για να κατανοήσετε τη ροή της διαδικασίας και να ενεργήσετε όταν οποιοδήποτε βήμα απαιτεί προσοχή.

### <a name="extensibility"></a>Επεκτασιμότητα

- **Εξαγωγή τμημάτων και άλλων δεδομένων στο Salesforce Marketing Cloud** Έχουμε επεκτείνει τους προορισμούς εξαγωγής μας, για να συμπεριλάβουμε το [Salesforce Marketing Cloud](export-salesforce.md). Τώρα μπορείτε να εξαγάγετε τμήματα και άλλους τύπους δεδομένων στο Salesforce Marketing Cloud μέσω μιας εξαγωγής με την επωνυμία SFTP. Η εισαγωγή δεδομένων μπορεί να αυτοματοποιηθεί πλήρως στο Salesforce και να χρησιμοποιηθεί για τη δημιουργία πιο αποτελεσματικών εκστρατειών μάρκετινγκ.  
 
- **Εξαγωγή τμημάτων στο ActiveCampaign** Έχουμε επεκτείνει στους προορισμούς εξαγωγής για να συμπεριλάβουμε την [Ενεργή εκστρατεία](export-active-campaign.md). Μπορείτε τώρα να εξαγάγετε τμήματα για να δημιουργήσετε εκστρατείες, να εκτελέσετε πρακτικές μάρκετινγκ μέσω ηλεκτρονικού ταχυδρομείου και να εργαστείτε με ειδικές ομάδες πελατών στο ActiveCampaign.
 
- **Εξαγωγή τμημάτων στο Sendinblue** Έχουμε επεκτείνει τους προορισμούς εξαγωγής μας για να συμπεριλάβουμε το [Sendinblue](export-sendinblue.md). Μπορείτε τώρα να εξαγάγετε τμήματα για να δημιουργήσετε εκστρατείες, να εκτελέσετε πρακτικές μάρκετινγκ μέσω ηλεκτρονικού ταχυδρομείου και να εργαστείτε με ειδικές ομάδες πελατών με το Sendinblue.
 
### <a name="ux-updates"></a>Ενημερώσεις UX 

- **Νέα και βελτιωμένη σελίδα "Πελάτες" και σελίδα λεπτομερειών προφίλ** Έχουμε επανασχεδιάσει τη σελίδα "Πελάτες" και τις σελίδες λεπτομερειών προφίλ για βελτιωμένη εμπειρία χρήστη και καλύτερη απόδοση. Αυτές οι αλλαγές σας επιτρέπουν να προβάλετε, να ταξινομήσετε, να αναζητήσετε και να φιλτράρετε πελάτες. Πλέον, στη διεύθυνση URL εμφανίζονται φίλτρα για την κοινή χρήση των αποτελεσμάτων αναζήτησης με άλλους χρήστες χωρίς προβλήματα. Τα αποτελέσματα αναζήτησης μπορούν επίσης να αποθηκευτούν ως τμήμα.    
  Τώρα, η σελίδα λεπτομερειών για τα προφίλ πελατών ομαδοποιεί δεδομένα σε διάφορες υπο-ενότητες, όπως δημογραφικά δεδομένα, αναγνωριστικά και άλλα χαρακτηριστικά προφίλ για βελτιωμένη αναγνωσιμότητα. Άλλες ενότητες στη σελίδα λεπτομερειών προφίλ είναι πλέον πιο αλληλεπιδραστικές. Για παράδειγμα, η ενότητα δραστηριοτήτων επιτρέπει πλέον το φιλτράρισμα και την ταξινόμηση.


## <a name="may-2021-updates"></a>Ενημερώσεις Μαΐου 2021

Οι ενημερώσεις για τον Μάιο 2021 περιλαμβάνουν διάφορες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="data-ingestion"></a>Κατάποση δεδομένων

- **Προβάλετε ή τροποποιήστε μεταδεδομένα ή ορισμούς οντοτήτων κατά την επισύναψη δεδομένων από το Azure Data Lake Storage**. Μπορείτε πλέον να προβάλλετε και να επεξεργαστείτε μεταδεδομένα ή ορισμούς οντοτήτων σε πληροφορίες κοινού κατά την επισύναψη δεδομένων από ένα φάκελο Κοινού μοντέλου δεδομένων στο δικό σας Azure Data Lake Storage. Αυτή η δυνατότητα παρέχει σχόλια σε πραγματικό χρόνο, επικύρωση μοντέλου και έλεγχο σφαλμάτων. Σας επιτρέπει να επεξεργάζεστε απρόσκοπτα τόσο το model.json όσο και το manifest.json.

### <a name="extensibility"></a>Επεκτασιμότητα

- **Βελτιωμένες εξαγωγές τμημάτων, προσαρμοσμένο χρονοδιάγραμμα και αναδιπλασιασμός** Μπορείτε πλέον να [δείτε όλες τις εξαγωγές για ένα συγκεκριμένο τμήμα](export-destinations.md#view-exports-and-export-details) σε μια λίστα. Αυτή η νέα προβολή βοηθά στη διαχείριση του τρόπου με τον οποίο χρησιμοποιείται ένα συγκεκριμένο τμήμα και στην προσαρμογή υπαρχουσών ή στη δημιουργία νέων εξαγωγών.    
  Μπορείτε να [καθορίσετε προσαρμοσμένα χρονοδιαγράμματα ανανέωσης](export-destinations.md#schedule-and-run-exports) για μεμονωμένες εξαγωγές ή πολλές εξαγωγές ταυτόχρονα. Έως τώρα, όλες οι εξαγωγές εκτελούνταν με κάθε ανανέωση του συστήματος.    
  Αντί να δημιουργήσετε μια νέα εξαγωγή από την αρχή, μπορείτε να ξεκινήσετε με βάση μια υπάρχουσα για να εξοικονομήσετε χρόνο.

- **Εξαγάγετε τμήματα στο Microsoft Advertising** Έχουμε επεκτείνει τους προορισμούς εξαγωγής για να συμπεριλάβουμε το Microsoft Advertising. Δημιουργήστε κοινά Customer Match στο Microsoft Advertising με τα ενοποιημένα δεδομένα προφίλ πελατών και χρησιμοποιήστε τα κοινά αυτά για τις διαφημιστικές εκστρατείες σας. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εξαγωγή τμημάτων στο Microsoft Advertising](export-microsoft-advertising.md).

- **Εξαγάγετε τμήματα σε διαφημίσεις LinkedIn** Έχουμε επεκτείνετε τους προορισμούς εξαγωγής μας, ώστε να συμπεριλάβουμε τις διαφημίσεις LinkedIn και να σας επιτρέπουμε να ξεκλειδώνετε τη στόχευση επαφών, καθώς και τη στόχευση εταιρείας μέσω LinkedIn εξάγοντας τα ενοποιημένα δεδομένα προφίλ πελατών σας. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εξαγωγή τμημάτων στο LinkedIn Ads](export-linkedin-ads.md).


- **Εξαγάγετε τμήματα στο Omnisend** Έχουμε επεκτείνει τους προορισμούς εξαγωγής για να συμπεριλάβουμε το Omnisend. Χρησιμοποιήστε τα τμήματα που δημιουργήθηκαν στις πληροφορίες κοινού για να δημιουργήσετε εκστρατείες, να παρέχετε μάρκετινγκ μέσω email και να αξιοποιήσετε συγκεκριμένες ομάδες πελατών με το Omnisend. Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εξαγωγή τμημάτων στο Omnisend](export-omnisend.md).

### <a name="predictions"></a>Προβλέψεις

- **Αναφορά χρηστικότητας δεδομένων εισόδου** Η αναφορά χρηστικότητας των δεδομένων εισόδου παρέχει μια ενοποιημένη προβολή των σφαλμάτων και των προειδοποιήσεων που ενδέχεται να δημιουργούνται από τις πρώτες σας προβλέψεις. Επίσης, παρέχει συστάσεις σχετικά με τον τρόπο βελτίωσης της απόδοσης του μοντέλου.    
  Η αναφορά είναι διαθέσιμη αφού ένα μοντέλο ολοκληρώσει τη διαδικασία εκπαίδευσης. Δημιουργείται για κάθε μοντέλο ξεχωριστά, ανεξάρτητα αν έχει ολοκληρωθεί με επιτυχία ή όχι.
  Αυτήν τη στιγμή, αυτή η δυνατότητα είναι διαθέσιμη μόνο για το μοντέλο Transaction Churn. Για περισσότερες πληροφορίες, ανατρέξτε στην [Αναφορά χρηστικότητας δεδομένων εισόδου](manage-predictions.md#input-data-usability-report).

### <a name="relationships"></a>Σχέσεις

- **Οπτικοποίηση σχέσης** Η προβολή οπτικοποίησης σχέσης σάς επιτρέπει να βλέπετε όλες τις υπάρχουσες σχέσεις μεταξύ των οντοτήτων και τον πληθάριθμό τους. Οι σχέσεις είναι πλέον οργανωμένες σε ομάδες: δημιουργία από τον χρήστη, το σύστημα και μεταβιβαζόμενες σχέσεις. Μπορείτε επίσης να εξαγάγετε μια προβολή ως εικόνα. Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα [Προβολή σχέσεων](relationships.md#view-relationships). 

## <a name="april-2021-updates"></a>Ενημερώσεις Απριλίου 2021

Οι ενημερώσεις του Απριλίου 2021 περιλαμβάνουν διάφορες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="data-unification"></a>Ενοποίηση δεδομένων
 
- **Βελτιωμένη εμπειρία συγχώνευσης για ενοποίηση δεδομένων**    
  
   Έχουμε τώρα μια βελτιωμένη εμπειρία χρήστη στη διαμόρφωση συγχώνευσης της διαδικασίας ενοποίησης δεδομένων. Οι αλλαγές περιλαμβάνουν διαισθητική ταξινόμηση των συγχωνευμένων πεδίων και λεπτομερή στατιστικά στοιχεία για πεδία συνδυασμένου και μεμονωμένου τύπου.

- **Ανασύνταξη οντότητας και ρύθμιση παραμέτρων όλων των εγγραφών προέλευσης στην οντότητα πελάτη**  
      
   Μπορείτε τώρα να αναδιατάσσετε και να καταργείτε οντότητες από ένα υπάρχον σχέδιο συνδυασμού στη διαδικασία ενοποίησης δεδομένων. Παρέχει ευελιξία για την αναδιάταξη των οντοτήτων στη διαδικασία αντιστοίχισης σύμφωνα με τις επιχειρηματικές ανάγκες. Επιπλέον, ενεργοποιούμε την ένταξη όλων των μη αντιστοιχιζόμενων εγγραφών στην τελική οντότητα *Πελάτη*, η οποία τους επιτρέπει να ορίζουν τον ορισμό συνόλου δεδομένων προφίλ πελατών.

### <a name="enrichments"></a>Εμπλουτισμοί

 - **Νέος εμπλουτισμός: Βελτιωμένες διευθύνσεις**    
  
   Είμαστε ενθουσιασμένοι που παρουσιάζουμε έναν νέο εμπλουτισμό για τη βελτίωση των διευθύνσεων στα δεδομένα των πελατών σας. Οι διευθύνσεις στα δεδομένα σας μπορεί να είναι μη δομημένες, ελλιπείς ή εσφαλμένες. Αυτή η δυνατότητα χρησιμοποιεί τα μοντέλα της Microsoft για να ομαλοποιήσετε και να εμπλουτίσετε τις διευθύνσεις σας στη μορφή Common Data Model για καλύτερη ακρίβεια και πληροφόρηση.
 
   Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εμπλουτισμός προφίλ πελατών με βελτιωμένες διευθύνσεις](enrichment-enhanced-addresses.md).

- **Καθοδηγούμενη εμπειρία διαμόρφωσης για εμπλουτισμούς**    
  
   Αναθεωρήσαμε την εμπειρία διαμόρφωσης για εμπλουτισμούς με μια απλή, καθοδηγούμενη εμπειρία. Τώρα έχετε μια σαφή, βήμα προς βήμα διαδικασία για τη δημιουργία και την επεξεργασία εμπλουτισμών.
 
   Επιπλέον, διαχωρίσαμε τη διαμόρφωση των συνδέσεων για εμπλουτισμούς τρίτων, για να επιτρέψουμε τη χρήση της ίδιας σύνδεση από πολλαπλούς εμπλουτισμούς. Μόνο οι διαχειριστές μπορούν να ρυθμίσουν τις παραμέτρους νέων συνδέσεων, αλλά οι δημιουργημένες συνδέσεις είναι διαθέσιμες τόσο στους διαχειριστές όσο και στους συμβάλλοντες.    

   Για περισσότερες πληροφορίες, ανατρέξτε στην ενότητα [Επισκόπηση συνδέσεων](connections.md).

- **Πολλαπλοί εμπλουτισμοί του ίδιου τύπου**    
  
   Τώρα επιτρέπουμε στους χρήστες να δημιουργούν και να διαχειρίζονται πολλαπλούς εμπλουτισμούς του ίδιου τύπου εμπλουτισμού. Για παράδειγμα, μπορείτε πλέον να δημιουργήσετε δύο ξεχωριστούς εμπλουτισμούς διευθύνσεων, για να εμπλουτίσετε δύο διαφορετικά τμήματα πελατών. Ισχύουν όρια για τον αριθμό των εμπλουτισμών του ίδιου τύπου που μπορούν να δημιουργηθούν και διαφέρουν ανάλογα με τον τύπο εμπλουτισμού.
  
   Για περισσότερες πληροφορίες, ανατρέξτε στο θέμα [Εμπλουτισμός για προφίλ πελατών](enrichment-hub.md).

## <a name="march-2021-updates"></a>Ενημερώσεις Μαρτίου 2021

Οι ενημερώσεις του Μαρτίου 2021 περιλαμβάνουν διάφορες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

### <a name="activities"></a>Δραστηριότητες

- **Οδηγός δραστηριότητας και σημασιολογικοί τύποι**

   Έχουμε βελτιώσει και ενημερώσει την εμπειρία αντιστοίχισης δραστηριοτήτων για να καθοδηγήσουμε και να απλοποιήσουμε τη δημιουργία αντιστοίχισης δραστηριοτήτων. Σε αυτήν τη νέα εμπειρία, οι χρήστες έχουν μια καθοδηγούμενη εμπειρία που σας βοηθά να ολοκληρώσετε κάθε βήμα της διαδικασίας. Στο βήμα αντιστοίχισης δραστηριοτήτων, εκτός από την επιλογή από πολλούς τύπους δραστηριοτήτων, ο χρήστης μπορεί να επιλέξει τη σημασιολογική αντιστοίχιση δεδομένων για τη *Συνδρομή* ή/και *SalesOrderLine* στα τυπικά σχήματα του κλάδου, τα οποία μπορούν να χρησιμοποιηθούν για κατάντη κατανάλωση.   

   Για περισσότερες πληροφορίες, δείτε το θέμα [Δραστηριότητες πελάτη](activities.md).

### <a name="data-ingestion"></a>Κατάποση δεδομένων

- **Συνδεθείτε με προελεύσεις δεδομένων εσωτερικής εγκατάστασης χρησιμοποιώντας ροές δεδομένων και πύλες Power Platform**. Πρόκειται να ανακοινώσουμε την προεπισκόπηση των ροών δεδομένων Power Platform και της συνδεσιμότητας εσωτερικής εγκατάστασης χρησιμοποιώντας πύλες στο Customer Insights με ένα συσχετισμένο περιβάλλον Power Platform ή Dataverse. Οποιεσδήποτε νέες προελεύσεις δεδομένων που δημιουργούνται σε ένα περιβάλλον Customer Insights με ένα συνδεδεμένο περιβάλλον Dataverse θα ορίζεται από προεπιλογή σε ροές δεδομένων Power Platform θέτωντας σε εφαρμογή τη συνδεσιμότητα δεδομένων εσωτερικής εγκατάστασης και ένα πλούσιο σύνολο συνδέσεων και δυνατοτήτων μετασχηματισμού.

Όλοι οι ρόλοι χρήστη έχουν πρόσβαση στη σελίδα **Εξαγωγές**, αλλά μόνο οι διαχειριστές μπορούν να επιλέξουν να επιτρέπουν στους συμβάλλοντες την επεξεργασία συγκεκριμένων εξαγωγών με κοινόχρηστες συνδέσεις.

## <a name="february-2021-updates"></a>Ενημερώσεις Φεβρουαρίου 2021

Οι ενημερώσεις του Φεβρουαρίου 2021 περιλαμβάνουν διάφορες δυνατότητες, αναβαθμίσεις επιδόσεων και επιδιορθώσεις σφαλμάτων.

#### <a name="extensibility"></a>Επεκτασιμότητα

- **Εξαγωγή τμημάτων στο AdRoll**

  Έχουμε επεκτείνει τους προορισμούς εξαγωγής για να συμπεριλάβουμε το AdRoll. Τώρα μπορείτε να εξαγάγετε τμήματα σε κοινά από το Customer Insights στο AdRoll και να τα χρησιμοποιήσετε ως γραμμή βάσης για τις διαφημίσεις σας. Για περισσότερες πληροφορίες, ανατρέξτε στη [Σύνδεση για AdRoll](export-adroll.md).

#### <a name="segments"></a>Τμήματα
 
- **Δημιουργία διπλοτύπου ενός τμήματος**
  
  Για να δημιουργήσετε ένα νέο τμήμα με βάση ένα υπάρχον, μπορείτε τώρα να δημιουργήσετε διπλότυπο τμήματος και να επεξεργαστείτε το διπλότυπο τμήμα για να το βελτιώσετε περαιτέρω. 

- **Προσθήκη επιπλέον χαρακτηριστικών σε τμήμα**

  Τώρα μπορείτε να συμπεριλάβετε χαρακτηριστικά στην έξοδο του τμήματός σας, ακόμα και αν αυτά τα χαρακτηριστικά δεν αποτελούν μέρος του προφίλ πελάτη. Για παράδειγμα, συμπεριλάβετε αναγνωριστικά συνδρομής σε ένα τμήμα, παρόλο που είναι μέρος της οντότητας συνδρομής που έχει σχέση M:1 με την οντότητα πελάτη. Εφόσον το χαρακτηριστικό ανήκει σε μια οντότητα που σχετίζεται με την οντότητα πελάτη, μπορείτε τώρα να συμπεριλάβετε αυτά τα χαρακτηριστικά.  

#### <a name="predictions"></a>Προβλέψεις

- **Δημιουργία προγνωστικών προτάσεων προϊόντων**

  Η κατανόηση του τι ενδιαφέρονται οι πελάτες να αγοράσουν είναι ένα από τα πρώτα βήματα που απαιτούνται για τη βελτίωση των επιχειρηματικών εσόδων και την ενίσχυση της αφοσίωσης των πελατών μέσω της εξατομίκευσης και της δέσμευσης. Η παροχή προτάσεων για προϊόντα που δεν είναι στοιχισμένα με τα ενδιαφέροντο του πελάτη σας μπορεί να δημιουργήσει μια αίσθηση αποσύνδεσης μεταξύ του πελάτη και της επιχείρησής σας και, εντέλει, να περιορίσει τα συνολικά πιθανά έσοδα και εμπειρία για έναν πελάτη. 

  Χρησιμοποιώντας τα δικά σας δεδομένα, μπορείτε πλέον να δημιουργήσετε προβλέψεις για τα προϊόντα που είναι πιθανό να αγοράσουν οι πελάτες σας στο μέλλον. Για περισσότερες πληροφορίες, ανατρέξτε στην [πρόβλεψη πρότασης προϊόντων](predict-product-recommendation.md).

  Οι διαχειριστές μπορούν να αντιγράφουν ρυθμίσεις παραμέτρων περιβάλλοντος σε ένα νέο περιβάλλον στον ίδιο οργανισμό. Αυτή η δυνατότητα επεκτείνει τη λειτουργικότητα του περιβάλλοντος αντιγραφής για περιπτώσεις στις οποίες χρησιμοποιούνται οι προελεύσεις δεδομένων που βασίζονται σε διαχειριζόμενη λίμνη δεδομένων Microsoft Dataverse ή σε φάκελο Common Data Model.
