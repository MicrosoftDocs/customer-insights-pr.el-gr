# <a name="enrich-customer-profiles-with-brand-and-interest-affinities-preview"></a>Εμπλουτίστε τα προφίλ πελατών με τις συγγένειες επωνυμίας και ενδιαφέροντος (προεπισκόπηση)

Χρησιμοποιήστε ιδιόκτητα δεδομένα της Microsoft για να εμπλουτίσετε τα δεδομένα των πελατών σας με συγγένειες επωνυμίας και συμφέροντος. Αυτές οι συγγένειες βασίζονται σε δεδομένα από άτομα με παρόμοια δημογραφικά στοιχεία με αυτά των πελατών σας. Αυτές οι πληροφορίες σας βοηθούν να κατανοήσετε καλύτερα και να χωρίσετε τους πελάτες σας με βάση τις συγγένειές τους με συγκεκριμένες επωνυμίες και ενδιαφέροντα.

Στις πληροφορίες κοινού, μεταβείτε στην επιλογή **Δεδομένα** > **Εμπλουτισμός** για να [ρυθμίσετε τις παραμέτρους και να προβάλετε εμπλουτισμούς](enrichment-hub.md).

Για να ρυθμίσετε τις παραμέτρους του εμπλουτισμού των συγγενειών επωνυμιών, μεταβείτε στην καρτέλα **Ανακάλυψη** και επιλέξτε **Εμπλουτισμός των δεδομένων μου** στο πλακίδιο **Επωνυμίες**.

Για να ρυθμίσετε τις παραμέτρους του εμπλουτισμού των συγγενειών ενδιαφέροντος, μεταβείτε στην καρτέλα **Ανακάλυψη** και επιλέξτε **Εμπλουτισμός των δεδομένων μου** στο πλακίδιο **Ενδιαφέροντα**.

   > [!div class="mx-imgBorder"]
   > ![Πλακίδια επωνυμιών και ενδιαφερόντων.](media/BrandsInterest-tile-Hub.png "Πλακίδια επωνυμιών και ενδιαφερόντων")

## <a name="how-we-determine-affinities"></a>Πώς θα καθορίσουμε τις συγγένειες

Χρησιμοποιούμε τα δεδομένα ηλεκτρονικής αναζήτησης της Microsoft για να βρείτε συγγένειες για επωνυμίες και συμφέρονται για διάφορα δημογραφικά τμήματα (που ορίζονται κατά ηλικία, φύλο ή τοποθεσία). Ο online όγκος αναζήτησης για μια εμπορική επωνυμία ή ένα ενδιαφέρον προσδιορίζει την συγγένεια ενός δημογραφικού τμήματος, σε σύγκριση με άλλα τμήματα, με αυτήν την εμπορική επωνυμία ή το ενδιαφέρον.

## <a name="affinity-level-and-score"></a>Επίπεδο συγγένειας και βαθμολογία

Σε κάθε εμπλουτισμένο προφίλ πελάτη, παρέχουμε δύο σχετικές τιμές: επίπεδο συγγένειας και βαθμολογία συγγένειας. Αυτές οι τιμές σάς βοηθούν να προσδιορίσετε πόσο ισχυρή είναι η συγγένεια για το δημογραφικό τμήμα αυτού του προφίλ, για μια επωνυμία ή ενδιαφέρον, σε σύγκριση με άλλα δημογραφικά τμήματα.

Το *επίπεδο συγγένειας* αποτελείται από τέσσερα επίπεδα και η *βαθμολογία συγγένειας* υπολογίζεται σε κλίμακα 100 σημείων που αντιστοιχεί στα επίπεδα συγγένειας.


|Επίπεδο συγγένειας |Βαθμολογία συγγένειας  |
|---------|---------|
|Πολύ υψηλό     | 85-100       |
|Υψηλός     | 70-84        |
|Μέτριος     | 35-69        |
|Χαμηλός     | 1-34        |

Ανάλογα με την λεπτομέρεια που θέλετε για τη μέτρηση της συγγένειας, μπορείτε να χρησιμοποιήσετε είτε το επίπεδο συγγένειας είτε τη βαθμολογία. Η βαθμολογία συγγένειας σας δίνει πιο ακριβή έλεγχο.

## <a name="supported-countriesregions"></a>Υποστηριζόμενες χώρες/περιοχές

Προς το παρόν, υποστηρίζουμε τις παρακάτω επιλογές χώρας/περιοχής: Αυστραλία, Καναδάς (Αγγλικά), Γαλλία, Γερμανία, Ηνωμένο Βασίλειο ή Ηνωμένες Πολιτείες (Αγγλικά).

Για να επιλέξετε μια χώρα ή περιοχή, ανοίξτε το στοιχείο **Εμπλουτισμός επωνυμιών** ή **Εμπλουτισμός ενδιαφερόντων** και επιλέξτε **Αλλαγή** δίπλα από την επιλογή **Χώρα/Περιοχή**. Στο τμήμα παραθύρου **Ρυθμίσεις χώρας/περιοχής**, κάντε μια επιλογή και επιλέξτε **Εφαρμογή**.

### <a name="implications-related-to-country-selection"></a>Συνέπειες που σχετίζονται με την επιλογή χώρας

- Όταν [επιλέγετε τις δικές σας μάρκες](#define-your-brands-or-interests), το σύστημα παρέχει προτάσεις με βάση την επιλεγμένη χώρα ή περιοχή.

- Όταν [επιλέγετε έναν κλάδο](#define-your-brands-or-interests), θα λάβετε τις πιο σχετικές επωνυμίες ή ενδιαφέροντα με βάση την επιλεγμένη χώρα ή περιοχή.

- Κατά τον [εμπλουτισμό των προφίλ](#refresh-enrichment), θα εμπλουτίσουμε όλα τα προφίλ πελατών για τα οποία περιλαμβάνονται δεδομένα για τις επιλεγμένες επωνυμίες ή τα επιλεγμένα ενδιαφέροντα, συμπεριλαμβανομένων των προφίλ που δεν βρίσκονται στην επιλεγμένη χώρα ή περιοχή. Για παράδειγμα, εάν επιλέξατε Γερμανία, θα εμπλουτίσουμε τα προφίλ που βρίσκονται στις Ηνωμένες Πολιτείες, εάν έχουμε διαθέσιμα δεδομένα για τις επιλεγμένες επωνυμίες και συμφέροντα στις Η.Π.Α.

## <a name="configure-enrichment"></a>Ρύθμιση παραμέτρων εμπλουτισμού

Μια καθοδηγούμενη εμπειρία σάς βοηθάει κατά τη ρύθμιση παραμέτρων των εμπλουτισμών. 

### <a name="define-your-brands-or-interests"></a>Καθορίστε τις επωνυμίες ή τα ενδιαφέροντά σας

Επιλέξτε έως και πέντε προϊόντα ή ενδιαφέροντα χρησιμοποιώντας μία ή και τις δύο επιλογές:

- **Κλάδος**: Επιλέξτε τον κλάδο σας από την αναπτυσσόμενη λίστα και, στη συνέχεια, επιλέξτε ένα μία από τις κορυφαίες επωνυμίες ή τα κορυφαία ενδιαφέροντα για τον εν λόγω κλάδο.
- **Επιλέξτε το δικό σας**: Εισαγάγετε μια εμπορική επωνυμία ή ένα ενδιαφέρον που είναι σχετικό με τον οργανισμό σας και, στη συνέχεια, επιλέξτε μια από τις προτάσεις που αντιστοιχούν. Εάν δεν παραθέσουμε μια εμπορική επωνυμία ή ένα ενδιαφέρον που αναζητάτε, στείλτε μας τα σχόλιά σας χρησιμοποιώντας τη σύνδεση **Πρόταση**.

### <a name="review-enrichment-preferences"></a>Προτιμήσεις εμπλουτισμού προεπισκόπησης

Ελέγξτε τις προεπιλεγμένες προτιμήσεις εμπλουτισμού και ενημερώστε τις ανάλογα με τις ανάγκες.

:::image type="content" source="media/affinity-enrichment-preferences.png" alt-text="Στιγμιότυπο οθόνης του παραθύρου προτιμήσεων εμπλουτισμού.":::

### <a name="select-entity-to-enrich"></a>Επιλογή οντότητας για εμπλουτισμό

Επιλέξτε **Εμπλουτισμένη οντότητα** και επιλέξτε το σύνολο δεδομένων που θέλετε να εμπλουτίσετε με δεδομένα εταιρείας από τη Microsoft. Μπορείτε να επιλέξετε την οντότητα Πελάτης για να εμπλουτίσετε όλα τα προφίλ πελατών σας ή να επιλέξετε μια οντότητα τμήματος αγοράς για να εμπλουτίσετε μόνο τα προφίλ πελατών που περιέχονται σε αυτό το τμήμα αγοράς.

### <a name="map-your-fields"></a>Αντιστοίχιση πεδίων

Αντιστοιχίστε πεδία από την ενοποιημένη οντότητα πελάτη για να ορίσετε το δημογραφικό τμήμα που θέλετε να χρησιμοποιεί το σύστημα για τον εμπλουτισμό των δεδομένων πελατών σας. Αντιστοίχιση χαρακτηριστικών χώρας/περιοχής και τουλάχιστον ημερομηνίας γέννησης ή φύλου. Επιπλέον, πρέπει να αντιστοιχίσετε τουλάχιστον ένα από τα στοιχεία "Πόλη" (και "Νομός/Επαρχία") ή "Ταχυδρομικός κώδικας". Επιλέξτε **Επεξεργασία** για να καθορίσετε την αντιστοίχιση των πεδίων και επιλέξτε **Εφαρμογή** όταν τελειώσετε. Επιλέξτε **Αποθήκευση** για να ολοκληρώσετε την αντιστοίχιση πεδίου.

Υποστηρίζονται οι παρακάτω μορφές και τιμές (στις τιμές δεν γίνεται διάκριση πεζών-κεφαλαίων):

- **Ημερομηνία γέννησης**: συνιστούμε η ημερομηνία γέννησης να μετατραπεί σε τύπο DateTime κατά τη διάρκεια της πρόσληψης δεδομένων. Εναλλακτικά, μπορεί να είναι μια συμβολοσειρά σε μορφή [ISO 8601](https://www.iso.org/iso-8601-date-and-time-format.html) "εεεε-ΜΜ-ηη" ή "εεεε-ΜΜ-ηηTωω:λλ:δδ".
- **Φύλο**: Άνδρας, γυναίκα, άγνωστο.
- **Ταχυδρομικός κώδικας**: Πενταψήφιοι ταχυδρομικοί κωδικοί για Ηνωμένες Πολιτείες, τυπικός ταχυδρομικός κώδικας οπουδήποτε αλλού.
- **Πόλη**: Όνομα πόλης στα Αγγλικά.
- **Νομός/Επαρχία**: Σύντμηση δύο γραμμάτων για τις ΗΠΑ και τον Καναδά. Σύντμηση δύο ή τριών γραμμάτων για την Αυστραλία. Δεν ισχύει για τη Γαλλία, τη Γερμανία ή το Ηνωμένο Βασίλειο.
- **Χώρα/Περιοχή**:

  - US: Ηνωμένες Πολιτείες της Αμερικής, Ηνωμένες Πολιτείες, ΗΠΑ, ΗΠΑ, Αμερική
  - CA: Καναδάς, CA
  - GB: Ηνωμένο Βασίλειο, ΗΒ, Μεγάλη Βρετανία, ΜΒ, Ηνωμένο Βασίλειο της Μεγάλης Βρετανίας και της Βόρειας Ιρλανδίας, Ηνωμένο Βασίλειο της Μεγάλης Βρετανίας
  - AU: Αυστραλία, AU, Κοινοπολιτεία της Αυστραλίας
  - FR: Γαλλία, FR, Γαλλική Δημοκρατία
  - DE: Γερμανία, γερμανικά, Deutschland, Allemagne, DE, Ομοσπονδιακή Δημοκρατία της Γερμανίας, Δημοκρατία της Γερμανίας

## <a name="refresh-enrichment"></a>Ανανέωση εμπλουτισμού

Εκτέλεση του εμπλουτισμού μετά τη ρύθμιση παραμέτρων των επωνυμιών, των ενδιαφερόντων και της αντιστοίχισης πεδίων για τα δημογραφικά στοιχεία. Για να ξεκινήσετε τη διεργασία, επιλέξτε **Εκτέλεση** στη σελίδα ρύθμισης επωνυμίας ή ενδιαφέροντος. Επιπλέον, μπορείτε να επιτρέψετε στο σύστημα την εκτέλεση του εμπλουτισμού αυτόματα ως μέρος μιας προγραμματισμένης ανανέωσης.

Ανάλογα με το μέγεθος των δεδομένων των πελατών σας, ενδέχεται να χρειαστούν αρκετά λεπτά για την ολοκλήρωση μιας εκτέλεσης εμπλουτισμού.

## <a name="enrichment-results"></a>Αποτελέσματα εμπλουτισμού

Μετά την εκτέλεση της διεργασίας εμπλουτισμού, μεταβείτε στο στοιχείο **Οι εμπλουτισμοί μου** για να εξετάσετε τον συνολικό αριθμό των εμπλουτισμένων πελατών και την ανάλυση των επωνυμιών ή των ενδιαφερόντων στο εμπλουτισμένο προφίλ πελατών.

:::image type="content" source="media/my-enrichments.png" alt-text="Προεπισκόπηση αποτελεσμάτων μετά την εκτέλεση της διαδικασίας εμπλουτισμού.":::

Εξετάστε τα εμπλουτισμένα δεδομένα επιλέγοντας **Προβολή εμπλουτισμένων δεδομένων** στο γράφημα. Τα εμπλουτισμένα δεδομένα για εμπορικές επωνυμίες πηγαίνουν στην οντότητα **BrandAffinityFromMicrosoft**. Τα δεδομένα για τα ενδιαφέροντα βρίσκονται στην οντότητα **InterestAffinityFromMicrosoft**. Επίσης, θα βρείτε αυτές τις οντότητες που περιλαμβάνονται στην ομάδα **Εμπλουτισμός** στα **Δεδομένα** > **Οντότητες**.

Θα δείτε ένα γράφημα με τον αριθμό των εμπλουτισμένων προφίλ πελατών με την πάροδο του χρόνου και μια προεπισκόπηση της εμπλουτισμένης οντότητας. Επιλέξτε **Εμφάνιση περισσότερων** στο πλακίδιο προεπισκόπησης για να ανοίξετε την εμπλουτισμένη οντότητα.

## <a name="see-enrichment-data-on-the-customer-card"></a>Δείτε τα δεδομένα εμπλουτισμού στην καρτέλα πελάτη

Επίσης, μπορείτε να προβάλλετε τις συγγένειες επωνυμίας και ενδιαφέροντος σε μεμονωμένες κάρτες πελατών. Μεταβείτε στην επιλογή **Πελάτες** και επιλέξτε ένα προφίλ πελάτη. Στην κάρτα πελάτη, θα βρείτε γραφήματα για τις επωνυμίες ή τα ενδιαφέροντα στα οποία τα άτομα στο δημογραφικό προφίλ αυτού του πελάτη έχουν συγγένεια.

:::image type="content" source="media/enrichment-customer-card.png" alt-text="Κάρτα πελάτη με εμπλουτισμένα δεδομένα.":::
